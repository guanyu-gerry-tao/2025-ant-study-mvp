# Project Ant

> 一个让你可以随时深究细节、又能回到主线的 AI 学习助手

## 📖 简介

Project Ant 是一个面向自学者的 iOS AI 学习助手，通过树状分支对话方式帮助用户攻克复杂知识点。

**核心特点**：
- 🌳 **树状对话**：像 Git 分支一样，随时深究概念而不打乱学习主线
- 🗺️ **路径可视化**：清晰展示你的知识探索路径
- 🎯 **上下文保持**：每个分支都继承完整的学习上下文
